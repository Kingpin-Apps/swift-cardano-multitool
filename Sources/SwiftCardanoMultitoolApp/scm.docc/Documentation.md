# ``SwiftCardanoMultitool``

A comprehensive interactive CLI for managing the Cardano blockchain ecosystem.

## Overview

`scm` (Swift Cardano Multitool) is an interactive terminal application covering the full Cardano operations lifecycle — installing node software, generating cryptographic keys, building and submitting transactions, querying live blockchain data, and running services.

The tool provides two interaction modes:

- **Interactive TUI** — run `scm` with no arguments for a guided, arrow-key–driven menu powered by the Noora terminal UI library.
- **Direct CLI** — pass commands and subcommands as arguments for scripting and automation (`scm query tip`, `scm install cardano-node`, etc.).
